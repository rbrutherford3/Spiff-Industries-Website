rename_images

This tool takes the metadata (EXIF) from image files and renames them based on their date, and vice versa. This allows the user to have a uniform format for their image library and also fill in missing EXIF data, where applicable.

Raspberry Pi

These two products were made for the Raspberry Pi operating system known as "Raspbian", which is now simply called "Raspberry Pi OS." This device is a great starting point for anyone new to programming, Linux, or embedded systems in general. It must be noted that these products are in no way affiliated with The Raspberry Pi Foundation.

initPi

This bash software is meant to run directly from the Raspbian Pi operating system. It makes setting up user accounts and security configurations easy, especially if you are using the Raspberry Pi in a "headless" (no monitor or keyboard) setup. The process is repeatable, so that one may easily reset their device back to a custom state when necessary (i.e.: bricking the system).

customNOOBS

This software may work in conjunction with initpi to create an automated Raspbian installation package. Basically, running the program prompts the user to indicate a bash or shell script to be run on the very first boot of the operating system. The program then creates a NOOBS (new out-of-the-box software) package by downloading the latest versions of Raspbian and NOOBS, unpacking root.tar.xz, and placing a one-time reference to the boot script in the rc.local file. All of this is to allow customization of the Raspbian experience.
